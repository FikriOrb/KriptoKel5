Extended Euclidean Algorithm

Polynomial inverse in GF(2)

Hitung inverse polinomial biner terhadap modulus AES atau modulus pilihan, lengkap dengan pembagian bersusun dan jejak koefisien setiap iterasi. 

r0 = 100011011
r1 = 01010011
while r1 != 0:
  q, r = div(r0, r1)
  t = t0 XOR q * t1
  r0, r1 = r1, r
Input

Cari inverse

Masukkan bit tertinggi lebih dulu. Contoh modulus AES: 100011011.

Modulus tidak boleh nol dan idealnya lebih panjang dari polinomial input.

Hanya digit biner 0 dan 1. Leading zero akan dinormalisasi.

Reference

Aturan GF(2)

Penjumlahan dan pengurangan polinomial sama-sama memakai XOR. Perkalian dilakukan dengan shift sesuai pangkat lalu dijumlahkan XOR. 

add(a, b) = a XOR b
subtract(a, b) = a XOR b
inverse(a) exists when gcd(modulus, a) = 1
